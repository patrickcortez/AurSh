# AurSh Contexts

A **Context** is basically a diskbacked object like data structure that can hold multiple attributes which are basically key value in nature. A **Context** can be invoked in the command line by simply doing:

```Aursh

echo <ContextName>:<AttributeName>


```

That would be replaced by the value of the attribute in that **Context**.
**Contexts** can be created, modified and deleted using: `aursh-context`.
